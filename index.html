Statements and Comments

Statement
Any line of a JS code is a statement like - a declaration statement, control statement... (you will learn about them later!)
Any statement in JS ends with a semicolon that is ';' unless you have a block! (you will learn about this later!)
Comment
Comment is a not actually a piece of code or maybe it can be defined as line of code which is ignored by the JS interpreter.
comments are very useful if you are working in a team or even if you are working alone. It helps to remember things.
There are two types of comments in JS - Single line comments and Multi line comments
Single line comments starts with double forward slashes (//) and Multi line comments are written within /* and */ as show below:

//this is a single line comment!
/* This is
a
multi line comment!
*/

Writing comments is regarded as a good practice in coding.

Variables and Data Types

DataTypes : In computer science and computer programming, a data type is a classification of data which tells the compiler or interpreter how the programmer intends to use the data. Most programming languages support various types of data, for example: real, integer or Boolean. That's written in wikipedia!
Common premitive date types available in programming languages are
int - Integer
float - Floating Point Numbers (Decimal Values)
boolean / bool - Boolean Value (true / false)
double - Big Floating Point Numbers
long - Big Integer Numbers

However, the plus point is that you don't have to write any data type name while declaring any variable in JS!

Varibales : In computer programming, a variable is a storage location paired with an associated symbolic name (an identifier), which contains some known or unknown quantity of information referred to as a value. That's the bookish definition!
While declaring a variable in JS you need to write var before it.

Varible declaration follows the rule of identifier -
  • Names can contain letters, digits, underscores, and dollar signs.
  • Names must begin with a letter
  • Names can also begin with $ and _ (but we will not use it in this tutorial)
  • Names are case sensitive (y and Y are different variables)
  • Reserved words (like JavaScript keywords) cannot be used as names

Declaring a varible

Look carefully at the code below :

Array

An array can be defined as a data structure that can store multiple values of a common data type.
An array is always stored in a contiguous block of memory.
Defining an array in javascript
Arrays is javascript can be delared as a variable but we can assign values to it inside square brackets.

var pets = ["dog", "cat", "rabbit"];

Also there is one more very popular way of declaring arrays that is, using the new keyword.
var pets = new array("dog", "cat", "rabbit");

Every element of array is stored with an index number. This index number is assigned automatically to each element in the array and in javascript it starts with 0,1,2,3,4....
Therefore, we can access any elements of an array very easily, like in the below code -
var pets = ["dog", "cat", "rabbit"];
document.write(pets[2]);

The above code will output rabbit.

Conditional Statements

We all make plans and decisions like - if this happens then I will do that else I'll do that!
We do same in coding with the help of conditional statements.

  • We use if to specify a block of code will be executed when a specific condition is true
  • We use else to specify a block of code will be executed when the if condition is false. else doesn't checks for any new condition
  • We use else if to speacify a new condition to test if the above condition is false

Now we are going to show you code examples of each of these control statements -
The if statement

var no_of_pencils = 19;
if(no_of_pencils < 15) {
document.write("you have less pencils");
}
The if statement with an else block
var no_of_pencils = 19;
if(no_of_pencils < 15) {
document.write("you have less pencils");
} else {
document.write("You have good ammount of pencils!");
}
The else if statement
var no_of_pencils = 19;
if(no_of_pencils < 15) {
document.write("you have less pencils");
} else if(no_of_pencils >=15 && no_of_pencils >20) {
document.write("You should buy more pencils!");
}
else {
document.write("You have good ammount of pencils!");
}

Loop statement in JavaScript

Loops are used to execute a block of code for a given time.
Loops are very useful if you have to run the same code few times, each time with a different value.

Different kinds of looping :

  • for - loops through a block of code a specific number of times
  • while - loops through a block of code while a specified condition is true
  • do/while - loops through a block of code while a specified condition is true. The only difference between do/while and while is that the code in do block get executed at least once.

Now we are going to look at code example of each of these loops -
The for loop

var i;
for(i = 0; i < 10; i++) {
document.write("The current value of I is" + i);
}

The above code will show the incremented value of i 10 times starting from 0.
The while loop
var i = 0;
while(i < 10) {
document.write("The current value of I is" + i);
i++;
}

The do while loop
var i = 0;
do {
document.write("The current value of I is" + i);
i++;
} while(i < 10);

Functions in JS

Function : A function can be defined as a block of reusable code that is designed to perform a particular task.
Functions provide better modularity for your application.
A function block is executed when "someone" or "something" call it (or invokes it).

A code of syntax for function in JS in shown below -
function myFun(x, y) {
return x + y;
}

The x and y which is inside paranthesis is called arguments. We pass arguments while invoking a function. They aren't necessary if our function don't have to perform any operation on a variable which is outside it's scope (we'll learn about this later).

The name convention of the function also follows the rules of name an identifier.
Different arguments are separated by commas.

How to invoke JS functions? : The function will execute when it gets invoked :
When an event occurs (when a user clicks a button)
Automatically (self invoked)
When it is invoked (called) from JavaScript code
Look at the below code carefully to learn how to invoke a function :

<script>
function myFun(x, y) {
return x + y;
}
var a = 5;
var b = 5;
var c = myFun(a, b);
document.write(c);
</script>

The above code will display the sum of these two variables in the browser.

The return statement : When code interpreter reaches the return statement of the function, the control goes out of the function returning a value.
Look at the below code :

<script>
function isEven(x) {
return (x%2 == 0);
}
var a = 8;
if(!isEven(a)) {
document.write("the number is odd!");
} else {
document.write("the number is
even!");
}
</script>

The above code has a function that returns a boolean value which is then checked by an if statement. If the value returned is true, it tells you that the number is even else, it tells you odd!

The biggest benefit of using function is that you can reuse the code as many times as you want!
If a variable is declared outside a function then it can't work inside the block of the function.
If a variable is declared inside a function, it can't do anything outside it OR it's scope is only till the closing brace.
